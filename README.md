# Small Business Security Toolkit

Practical, jargon-free security setup guides and templates for small businesses and non-technical teams.

Built by TrustGrid Technology Limited.

## Why This Exists

Most cybersecurity resources are written for enterprise security teams with dedicated IT departments. Small businesses do not have that. A 3-5 person team running a roofing company, an e-commerce shop, or a consulting practice needs security that is simple, practical, and maintainable without ongoing technical support.

This toolkit is built specifically for that gap.

## Templates Included

| Template | Description |
|---|---|
| Password Manager Selection Guide | Comparison of password managers suited to small teams |
| Credential Sharing Protocol | Secure way to share access with remote staff and VAs |
| Access Provisioning Checklist | What to do when someone joins, leaves, or changes roles |
| 2FA Rollout Checklist | Step by step guide to enabling two-factor authentication |
| Plain English Security Guide Template | A 1-3 page guide template your team can actually use |
| Small Business Risk Quick Assessment | Fast risk review to flag your highest priority gaps |

## Who This Is For

- Small businesses with 2-20 employees
- Teams without a dedicated IT or security person
- Businesses working with remote staff, virtual assistants, or offshore teams
- Founders who know security matters but do not know where to start

## Principles Behind This Toolkit

1. Simple beats comprehensive. A security practice nobody follows is worthless.
2. Document everything in plain English. No jargon, no acronyms without explanation.
3. Build for the person who will use it, not the person who built it.
4. Security should reduce friction over time, not add to it permanently.
